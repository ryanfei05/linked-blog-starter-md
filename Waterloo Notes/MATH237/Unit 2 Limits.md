**Proving a Limit DNE**
- show a Limit DNE by approaching (a,b) along any smooth curve, and if you get 2 different limits, use the fact that the limit must be unique -> therefore DNE
- this can only be used to show that limit **DNE**, not existence
Try:
- y = 0
- x = 0
	- when approaching along x or y axis, the limit becomes a limit in 1D (use techniques from 1D calc) (find left hand != right hand limits)
- y = x
- y = mx
	- arbitrary coefficient m. And if the value depends on m -> not unique -> DNE (since we can choose any m leading to diff values of limit)
- y = mx^2 
	- try values where x in numerator and denominator can cancel
